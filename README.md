# Decentralized Voting Application - Server

## Installation
```
npm install -g truffle
npm install -g ganache-cli
npm install -g solium
```

## Compile smart contract
```
truffle compile
```

## Start EVM with Ganache CLI
```
ganache-cli
```

## Deploy smart contract
```
truffle migrate
```
